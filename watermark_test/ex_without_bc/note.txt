[ 워터마킹 성능 지표 설명 ]

■ 지표별 범위 및 의미
| 지표              | 범위 (이상적)  | 설명 |
|------------------|----------------|------------------------------------------------|
| PSNR             | → ∞ dB         | 화질(왜곡). 30 dB↑이면 사람이 잘 못 느낌       |
| SSIM             | 0 → 1          | 구조적 유사도 (1 = 동일)                        |
| Bit Accuracy     | 0 → 1          | 삽입 비트 중 정확히 복호된 비율                 |
| BER              | 1 - Accuracy   | 비트 오류율                                     |
| NC               | 0 → 1          | 패턴 상관계수 (1 = 완벽)                        |

■ PSNR (dB) 기준 품질 등급
| PSNR (dB)       | 품질 등급           | 설명 |
|------------------|----------------------|----------------------------------------------|
| > 40 dB          | 매우 우수 (거의 무손실) | 사람이 차이를 거의 느낄 수 없음                 |
| 35–40 dB         | 우수                 | 품질 저하 거의 없음                             |
| 30–35 dB         | 양호                 | 약간의 품질 저하 (대부분 허용 가능)              |
| 28–30 dB         | 보통                 | 워터마크 삽입이 눈에 띌 수 있음                  |
| < 28 dB          | 저품질               | 시각적으로도 품질 저하가 확연함                   |

■ 각 지표 해석 요약
| 지표     | 높을수록 좋음? | 설명 |
|----------|----------------|------------------------------------------------------|
| PSNR     | ↑             | Stego 이미지가 원본과 유사함 (왜곡 적음)              |
| SSIM     | ↑             | 구조적 유사도 (사람 눈 기준 품질)                     |
| ACC      | ↑             | 워터마크 비트 추출 정확도 (100%면 완전 추출 성공)       |
| BER      | ↓             | 워터마크 비트 오차율 (0%면 완벽 추출)                 |
| NC       | ↑             | 정답 비트와 추출 비트 간 상관관계 (1이면 완전 일치)     |

[ 워터마킹 성능 판단 기준 요약 ]

■ 좋은 성능으로 판단할 수 있는 기준

| 지표              | 기준 값                 | 설명 |
|------------------|--------------------------|------------------------------------------------|
| PSNR             | 30 dB 이상               | 왜곡이 적고, 사람이 보기에도 품질 저하가 거의 없음 |
| SSIM             | 0.95 ~ 1.0               | 구조적으로 원본과 매우 유사 (사람 눈 기준)        |
| Bit Accuracy     | 0.95 ~ 1.0               | 삽입한 워터마크 비트를 거의 완벽히 복원 가능      |
| BER              | 낮을수록 좋음            | Bit Accuracy에 따라 자동으로 결정됨               |
| NC               | 1.0에 가까울수록 좋음     | 정답 비트와의 상관관계 (Accuracy가 높으면 NC도 높음) |

※ BER (Bit Error Rate) = 1 - Bit Accuracy  
※ NC (Normalized Correlation)은 Accuracy 기반이므로 부가 지표로 간주 가능


■ 최종 요약

아래 세 가지 조건을 동시에 만족하면,  
워터마킹 성능이 시각적으로도 기술적으로도 **우수**하다고 평가할 수 있음:

    - PSNR ≥ 30 dB
    - SSIM ≈ 1
    - Bit Accuracy ≈ 1.0

→ BER, NC는 보조적으로 해석 (Bit Accuracy에 의존)



[ 워터마킹 공격 종류 및 영향 ]

| 공격             | 설명                  | 파라미터 예시                                 | 일반적 영향                             |
|------------------|-----------------------|----------------------------------------------|------------------------------------------|
| JPEG             | 손실 압축 포맷으로 품질 저하    | JPEG_Q = [50~95] (낮을수록 압축 강함)              | 블록 노이즈, 세부 손실                       |
| Gaussian Blur    | 흐리게 만들기 (시각적 흐림 효과) | GB_SIG = [2~7] (σ 값이 클수록 흐려짐)               | 경계 정보 손실                              |
| Resize           | 이미지 축소 후 복원           | RS_SCALES = [0.5~0.9] (작게 줄였다가 되돌림)        | 픽셀 단순화, 디테일 손실                      |
| Crop             | 중앙 부분만 자르고 리사이즈     | CR_PCTS = [0.6~0.9] (60%만 보존 등)                 | 가장자리 정보 손실                            |
| Gaussian Noise   | 랜덤 노이즈 추가              | GN_SIGMA = [0.01~0.03] (표준편차 σ)               | 무작위 점 찍힘 (salt & pepper처럼 보임)         |
